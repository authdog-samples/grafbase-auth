# GraphQL Group based access Control with Grafbase and Authdog

## Components

- Console (default port: `3000`)
- Backend API (default port: `4000`)

## Requirements

- `pnpm` (https://pnpm.io/installation)

## Get Started

### 1) Install dependencies

```bash
make install
```

### 2) Start dev environment

```bash
make dev
```
